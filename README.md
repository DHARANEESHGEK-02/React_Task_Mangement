# 🚀 Installation Guide

### 1️⃣ Clone Repository
```bash
git clone https://github.com/DHARANEESHGEK-02/Task_Management_React.git
```

### 2️⃣ Navigate to Project
```bash
cd Task_Management_React
```

### 3️⃣ Install Dependencies
```bash
npm install
```

### 4️⃣ Run Development Server
```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
